SAANS Data Capture App (Vite + React + Tailwind)

How to run locally:

1. Unzip the project.
2. Install dependencies:
   npm install
3. Run the dev server:
   npm run dev
4. Open the URL shown by Vite (usually http://localhost:5173)

To build for production:
  npm run build

This project includes TailwindCSS. If you deploy to Vercel, it will auto-detect the Vite build step.
